# Círculo Trigonométrico Interactivo

Aplicación web interactiva para el aprendizaje del círculo trigonométrico.

## Características
- Visualización dinámica del ángulo
- Cálculo de seno, coseno y tangente
- Interfaz educativa
- Uso en nivel secundario y universitario
